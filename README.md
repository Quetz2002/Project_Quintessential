#Project Quintessential#

*Este es un proyecto de Multijugador en línea cuya idea principal es realizar un juego de terror y supervivencia donde los jugadores puedan sobrevivir y escapar de una zona en concreto.*

##Implementación##

*La implementación del multijugador se hara utilizando los paquetes de NetCode for GameObjects de Unity y el Multiplayer Play Mode los cuales ayudaran al desarrollo del proyecto en si, logrando de esta forma instanciar dos jugadores*
*independientes usando un mismo código aunque para esto se cambia la herencia de MonoBehavior a NetworkBehavior utilizando una lógica para verificar que el comportamiento si pertenezca al jugador que debe pertenecer o no para que solo se mueva el que debe de moverse*
